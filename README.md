# Password Tool CLI

## About

This is a simple Python command line tool that allows users to check the strength of a password or generate a strong random password.

The project was built to practice CLI arguments, basic validation, and random password generation.

## Features

- Check password strength
- Generate strong random passwords
- Single-file implementation
- No external dependencies

## Rules

- Password is passed directly through the command line
- Strength is based on basic rules only
- Generated passwords include letters, numbers, and symbols
- No data is stored or logged

## Usage
- python password_tool.py check MyPassword@123
- python password_tool.py generate 12
